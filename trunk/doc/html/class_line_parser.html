Detailed Description

Class for parsing file line by line.

The idea of this class is coming from another open project called "Back to roots" http://sourceforge.net/projects/backtoroots/ and is under GPL copyright: Copyright (C) 2010 BEYLER Jean Christophe

With this class you can parse:

  • bool
  • int
  • float
  • IVec2
