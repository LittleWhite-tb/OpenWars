Detailed Description

Base for the API specific renderer.

Give an unified interface usable by the application, to draw on the screen This interface has to be inherited to implement a new API specific renderer (SDL / OpenGL / ...) A Window instance is stored in this class to know on which window it has to draw The following functionnalities needs to be implemented:

  • Clear the screen
  • Draw a rectangle with a unified colour
  • Draw a Sprite
  • Draw a part of a Sprite
  • Draw an AnimatedSprite
  • Draw an image in background
