Detailed Description

Scaling operation utilities.

Give functions to handle the sprites scaling operations The scale factor is determined by using the size of the window and some globals (see globals.h) The scale functions is inspired by the SDL_gfx library setScaleFactor() neeeds to be call after the Window creation, but before the SpriteManager usage (so before creating a Sprite or AnimatedSprite)
