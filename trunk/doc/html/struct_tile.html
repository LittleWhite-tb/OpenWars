Detailed Description

Tile struct.

Keeps the following:

  • The enum to characterise the tile (id)
  • The defence of the tile (gameplay)
  • If the tile is a see tile (gameplay)
  • If the tile is a building (gameplay)
  • The life of the building (gameplay)
